# Accessing configuration

The following code snippet shows how to access the configuration object from the `theme.config.js` file.

```


import Config from '../../theme.config'


## Accessing project.link

{Config.project.link}

## Accessing logo

{Config.logo}

## Accessing chat

### Accessing chat.link

{Config.chat.link}

### Accessing chat.icon

{Config.chat.icon}
```
