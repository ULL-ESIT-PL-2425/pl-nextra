## Parse::Eyapp

Parse::Eyapp is an extension of yacc. 
It provides a compiler which lets you compile yacc -like grammars into a Perl LALR(1) Object Oriented parser. 
It automates the building of the abstract syntax trees,  
provides an API for tree transformation and has being used inside the Perl community in many projects 
(see for instance the RPerl compiler developed by AutoParallel Technologies at http://rperl.org/). 

Casiano Rodriguez-Leon created the compiler and maintained during the years 
2006 to 2017. 
From 2017 up to now is maintained by William N. Braswell, Jr.  

See https://metacpan.org/dist/Parse-Eyapp/view/eyapp, 
https://cpan.metacpan.org/authors/id/C/CA/CASIANO/, 
https://metacpan.org/author/CASIANO/releases
