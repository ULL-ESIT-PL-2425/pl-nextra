I notice many of you are having a hard time trying to publish the module to the GitHub Package Registry.

- How to get the GitHub token at minute 19:58
- Giving the token to the npm client at minute 23:00
- `npm publish` at minute 23:52
- Associating the GitHub Organization ULL-ESIT-PL-2324 as a scope to the github registry at minute 28:13
